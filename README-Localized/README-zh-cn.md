# Microsoft Graph：验证和连接到 Microsoft Graph

此模块演练 Microsoft Graph 的身份验证场景。

## 实验

在此实验中，你将使用 Graph SDK 和 Microsoft 身份验证库 (MSAL) 演练采用 Microsoft Graph 的身份验证和权限场景。

- [实验手册](./lab.md)
